# 可不可愛、日文厲害（Japanese_learning）

可愛像素風 RPG 日文學習遊戲。玩家扮演誤闖日文像素世界的勇者，用日文當武器打怪升級。針對**完全新手到 N1** 設計：從動漫常用語與五十音出發，經旅遊實戰句、NPC 會話劇情，一路打到 N1 魔王城。

## 怎麼跑

無 build、無依賴，直接用瀏覽器開 `index.html` 即可（建議手機或視窗縮窄成直式）。

- 需要網路嗎：不需要。全部素材（像素圖、音效）皆以 Canvas / WebAudio 程式合成。
- 日文發音：使用瀏覽器內建 TTS（`speechSynthesis`，ja-JP）。裝置沒有日文語音時，聽力題會自動退化為閱讀題。
- 進度：存在 `localStorage`（key：`kawaii_quest_save_v1`），標題畫面可重置。

## 目前狀態（v0.5，2026-07-11）

**世界地圖 10 大區域、74 章、443 個學習項目**，區域逐步解鎖（前一區通關半數章節開下一區）。JLPT 順序：N5 最簡單 → N1 最難，遊戲依此由 N5 草原排到 N1 魔王城：

| 區域 | 內容 |
|---|---|
| 新手村 | 動漫常用語＋寒暄（13 句） |
| 五十音之森 | 清音全 10 行＋濁音・半濁音・拗音，共 80 假名，每字附字源聯想＋代表例字（卡背可點🔊唸例字，寫字板同步顯示） |
| 旅人驛站 | 旅遊實戰句 5 主題（移動/餐廳/購物/旅館/緊急） |
| 月見酒館 | Visual Novel 會話劇情 4 場（選錯會解釋為什麼失禮） |
| 歡樂祭典（11 章） | 趣味主題：美食祭×2/旅遊景點/日本文化/自我介紹/職業/私生活/戀愛單字/告白台詞/動漫用語/冷笑話祭（日本諧音梗附笑點解說） |
| N5 草原（8 章） | 數字/時間/生活名詞/動詞/形容詞/家族/食物/身體顏色 |
| N4 街道（6 章） | 動詞進階/城市生活/心情形容/學校工作/副詞/自然天氣 |
| N3 港町（6 章） | 社會生活/動詞/心情狀態/人際關係/副詞/生活動詞（漢字＋注音） |
| N2 雪山（5 章） | 抽象名詞/動詞副詞/動詞/形容詞/慣用表現 |
| N1 魔王城（4 章） | 魔王語彙壹～參/副詞篇，門番龍到日語龍神鎮守 |

各級為精選核心字彙（非完整考綱字表），資料層 `vh()`/`vk()` 一行一字可持續擴充。

**玩法系統**：
- 學習翻卡（TTS 發音＋聯想記憶）／限時戰鬥（讀題＋聽力混出，快答觸發 CRITICAL，答錯排回隊伍）
- **主題魔王戰**：每區最後一章，需先通關全區才解鎖；出題池＝全區內容隨機抽 12 題、時限縮短為 8 秒——打贏＝該主題畢業考通過
- RPG 養成：EXP／等級／稱號／武器進化（見習木劍 → 魔王討伐聖劍，7 階）
- Roguelike 祝福（勝利後三選二抽一）、畫符文寫字板（假名章節，粒子特效）、8-bit WebAudio 音效
- 戰鬥可「🏳 逃走」無懲罰返回
